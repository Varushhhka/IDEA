# IDEA
 Реализация алгоритма симметричного блочного шифрования IDEA в режимах ECB и OFB на языке программирования Java
Состоит из нескольких файлов: 
1.	Idea.java – реализует ядро алгоритма. Содержит в себе необходимые математические функции, генерацию ключей зашифрования/расшифрования, функцию, выполняющую полный процесс шифрования и вспомогательный метод преобразования строчного пароля в 128-битовый ключ, пригодный для использования в алгоритме; 
2.	IdeaFileEncryption.java – позволяет работать с файлами. Он объединяет логику чтения данных из исходного файла, их зашифрование/расшифрование с использованием ядра (Idea.java) и запись результата в выходной файл;
3.	BlockStreamCrypter.java – реализует логику выбранного режима работы блочного шифра (ECB или OFB);
4.	IdeaCmd.java – вспомогательный файл, необходимый для взаимодействия с программой через консоль;
5.	TestIdeaMath.java – тесты по методу «белый» ящик, они верифицируют каждую математическую операцию отдельно, что гарантирует отсутствие ошибок в фундаментальных компонентах алгоритма.;
6.	TestIdeaVectors.java – тесты по методу «черный» ящик, они проверяют соответствие результатов работы программы официальным тестовым векторам, подтверждая корректность шифрования и расшифрования.
